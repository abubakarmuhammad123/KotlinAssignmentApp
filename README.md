# My Kotlin Android App

This is a simple Android app created using **Kotlin** in **Android Studio**.  
The app demonstrates:

- A basic **Empty Activity** layout
- A **TextView** showing a message
- A **Toast** message when the app starts

## Project Structure

- `app/src/main/java` → Kotlin code (`MainActivity.kt`)  
- `app/src/main/res` → Layouts and resources (`activity_main.xml`)  
- `build.gradle.kts` & `settings.gradle.kts` → Gradle build files  

## How to run

1. Open this project in **Android Studio**.  
2. Sync Gradle if required.  
3. Run the app on an emulator or real device.
